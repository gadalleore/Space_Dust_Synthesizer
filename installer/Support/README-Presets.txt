Space Dust — Presets folder
============================

This directory is where Space Dust looks for your preset files (.xml).

Adding presets
--------------
- Save presets from the plug-in UI into this folder, or copy compatible .xml
  preset files here.
- Restart the host or rescan plug-ins if your DAW caches preset lists.

Backing up
----------
Copy this entire folder to keep your sounds safe before OS upgrades or
reinstalls.

Need help?
----------
Preset location is stored in the Space Dust configuration file written by the
installer. To change the folder later, reinstall and choose a new path, or edit
the configuration as described in the product documentation.
